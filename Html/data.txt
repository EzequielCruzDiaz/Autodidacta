# ascii: ASCII (American Standard Code for Information Interchange) es un conjunto de caracteres basado en el alfabeto inglés que asigna un valor numérico único a cada símbolo, 
como letras, números, signos de puntuación y caracteres especiales.

# meta elements: en HTML, las etiquetas meta proporcionan metadatos sobre un documento HTML, 
como el autor, la descripción del contenido, palabras clave relevantes, el conjunto de caracteres utilizado, etc.

# elemento <p></p> sirve para crear parrafos.

# elemento <em></em> sirve para hacer enfasis en una palabra o un bloque de texto especifico.

# elemento <strong> Sirve volver negrita una palabra o un bloque de texto en especifico.

# existen 6 elementos <h1>....<h6> los cuales sirven para crear headers 

# elemento <a> se crean los enlaces para direccionar al usuario. ej:<a href="mailto:ezequielcruz018@gmail.com">Send me a message</a>

# elemento object-fit: modifica los bordes de los elementos en una pagina.

# elemento <video>: insertar video, en conjunto de la propiedad controls, para adaptar el video.

# elemento <audio>: insertar audio,en conjunto de la propiedad controls, para adaptar el sonido.

#elemento Hay tres tipos principales de listas en HTML: 
listas desordenadas (<ul>), 
listas ordenadas (<ol>), 
listas de definición (<dl>)

# elemento <li> significa un elemento en la lista.

# elemento li*3 crea automaticamente 3 elementos de lista.

# elemento ol>li*3 crea automaticamente toda la lista y sus espacios.

# elemento tablas: Las tablas en HTML son utilizadas para mostrar datos de manera organizada y estructurada en forma de filas y columnas.
<table>: Define una tabla.
<tr>: Define una fila en la tabla.
<td>: Define una celda de datos en una fila.
<th>: Define una celda de encabezado de columna o fila.

# Atributos principales:
border: Establece el ancho del borde de la tabla.
cellpadding: Espacio interior entre el borde de la celda y su contenido.
cellspacing: Espacio entre las celdas de la tabla.
colspan: Especifica el número de columnas que una celda debería ocupar.
rowspan: Especifica el número de filas que una celda debería ocupar.

 # Encabezados y cuerpos de tabla:

<thead>: Define el encabezado de una tabla.
<tbody>: Define el cuerpo de una tabla.
<tfoot>: Define el pie de una tabla
